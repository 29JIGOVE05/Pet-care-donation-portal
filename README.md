# Paws and Whiskers Shelter

A simple static website created to support stray animals by promoting food drives, medical aid, adoption, and donations.

## Technologies Used

HTML

CSS

JavaScript

## Features

Informative sections about the shelter

Food drive, medical aid, and adoption details

Donation button with payment link

Simple and user-friendly design

## How to Run

Open index.html in any web browser.
